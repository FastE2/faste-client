# рџљЂ Getting Started

## Requirements

* Node.js >= 20
* npm / pnpm / yarn

## Installation

```bash
git clone https://github.com/ahkiet22/faste-client.git

cd faste-client

npm install
```

Create your environment file:

```bash
cp .env.example .env
```

Run development server:

```bash
npm run dev
```

Production build:

```bash
npm run build
```

Start production:

```bash
npm run start
```

Storybook:

```bash
npm run storybook
```
